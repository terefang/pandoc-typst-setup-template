# Supported Syntax

the base markdown syntax is pandocs with the following options:

* +pipe_tables
* +raw_attribute
* +raw_html
* +fenced_divs
* +yaml_metadata_block
* -tex_math_dollars

## markdown extensions

### pipe-tables

#### col-spans

if a cell consists solely of either the character "`<`" or "`>`" then the cell is merged (spanned) with the pointing cell.
